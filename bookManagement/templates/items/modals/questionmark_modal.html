Search functionality

Choose parameter from parameter combo box then:

For 'Title': Write down title of the book. For example: 'Pan Tadeusz'

For 'Author(s)': Write down author(s) name. For example: 'Adam Mickiewicz'.

For 'Language': Write down language of the book. For example: 'Polish'.

For 'Publish date exact': Write down or choose from calendar exact published date of the book. For example: '1834-10-30'

For 'Publish date range': In left input write down or choose date from when you want to search. In right input write down or choose date to when you want to search. For example: 'yyyy-mm-dd to yyyy-mm-dd'. For example: '1834-01-01 to 1834-12-31'.
